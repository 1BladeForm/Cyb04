# Сетевые и безопасностные операции

## Задачи:

### 1. Установка SSH-сервера на Windows 10/Windows Server
1. **Описание**: Установите и запустите SSH-сервер на виртуальной машине с Windows 10 или Windows Server.
2. **Шаги**:
    - Откройте PowerShell с правами администратора.
    - Введите команду: `Add-WindowsCapability -Online -Name OpenSSH.Server`.
    - Запустите SSH-сервер с помощью команды: `Start-Service sshd`.
    - Убедитесь, что SSH-сервер настроен для автоматического запуска: `Set-Service -Name sshd -StartupType 'Automatic'`.
3. **Скриншот**:
    ![Установка и запуск SSH-сервера](https://i.imgur.com/JviJPBW.png)

### 2. Разрешение подключения по RDP
1. **Описание**: Настройте удаленный доступ к виртуальной машине через Remote Desktop Protocol (RDP).
2. **Шаги**:
    - Откройте **Панель управления** > **Система и безопасность** > **Система**.
    - Выберите **Настройка удаленного доступа**.
    - Включите параметр **Разрешить удаленные подключения к этому компьютеру**.
3. **Скриншот**:
    ![Настройка RDP](https://i.imgur.com/Dtjd1h9.png)

### 3. Сканирование подсети с использованием Kali Linux
1. **Описание**: Выполните сканирование хоста с использованием Kali Linux.
2. **Шаги**:
    - Откройте терминал на Kali Linux.
    - Используйте команду `nmap` для сканирования подсети: `nmap -Pn 10.10.0.4`.

### 5. Проведение BruteForce-атаки на SSH-сервер на Windows 10
1. **Описание**: Проведите BruteForce-атаку на SSH-сервер, установленный на виртуальной машине с Windows 10, используя инструменты Kali Linux.
2. **Шаги**:
    - Убедитесь, что SSH-сервер работает и доступен для подключения.
    - Используйте утилиту `hydra` для выполнения BruteForce-атаки: `hydra -l adminisrator -P wordlists.txt ssh://10.10.0.4>`.
